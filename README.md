# CS2 Bind Checker

Простая программа на Python, которая автоматически находит установленный Steam и Counter-Strike 2, читает файлы конфигурации игрока и отображает его настройки.

## Возможности

* 🔍 Автоматически находит установленный Steam.
* 🎮 Находит все аккаунты Steam с установленной CS2.
* ⌨️ Показывает все бинды клавиш.
* 📋 Добавляет стандартные бинды CS2, если они отсутствуют в конфигурации.
* ⚙️ Показывает пользовательские бинды, заменяющие стандартные.
* 🎯 Отображает чувствительность мыши (Sensitivity).
* 💻 Работает без необходимости вручную указывать путь к игре.
* 📦 Доступен исходный код и готовый `.exe`.

## Требования

* Windows 10 / 11
* Установленный Steam
* Установленная Counter-Strike 2

## Использование

### Вариант 1 — готовый `.exe`

1. Скачайте последнюю версию из раздела **Releases**.
2. Запустите `CS2 Bind Checker.exe`.
3. Программа автоматически найдёт Steam и выведет информацию о биндах и настройках.

### Вариант 2 — запуск из исходного кода

1. Установите Python 3.12 или новее.
2. Скачайте исходный код.
3. Запустите:

```bash
python main.py
```

## Что отображает программа

* Все стандартные бинды CS2.
* Все изменённые пользователем бинды.
* Чувствительность мыши (Sensitivity).
* Найденный путь к папке конфигурации CS2.

## Компиляция в EXE

```bash
pip install pyinstaller
pyinstaller --onefile main.py
```

Готовый файл появится в папке:

```
dist/
```
