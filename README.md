# CS2 Bind Checker

A simple Python tool that automatically finds the installed Steam and Counter-Strike 2 directories, reads the player's configuration files, and displays the current game settings.

## Features

* Automatically detects the Steam installation.
* Searches for all CS2 user profiles.
* Displays all key bindings.
* Includes default CS2 key bindings if they are not explicitly saved in the configuration.
* Shows custom key bindings that override the default ones.
* Displays mouse sensitivity.
* No manual configuration required.
* Can be compiled into a standalone `.exe` using PyInstaller.

## Requirements

* Windows
* Steam
* Counter-Strike 2

## Usage

1. Download the latest release or run the Python script.
2. Launch the program.
3. The tool will automatically locate your CS2 configuration and display your key bindings and mouse sensitivity.
